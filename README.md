## 项目介绍

这个项目是我用 Java 写的一个**简易智能体通信系统**，主要用来模拟多个智能体之间的消息发送、接收和处理过程，同时贴合 ACPs 协议的基本思想。
------

## 代码实现了什么

1. **创建两个智能体**，一个主智能体、一个工作智能体
2. **智能体可以互相发送消息**，并自动进入工作状态
3. 消息会经过**简单加密、解密、格式校验**
4. 智能体有自己的状态：空闲、工作、忙碌、休眠、异常
5. 使用**消息队列**保证消息不乱序、不丢失
6. 支持**多线程运行**，两个智能体同时工作
7. 内置小工具：时间格式化、性能监控、日志输出等
8. 程序可以安全启动、运行、关闭

## 运行效果

程序启动后，两个智能体会：

- 互相打招呼
- 查询状态
- 执行同步任务
- 打印日志、耗时、处理结果

整个过程能清晰看到智能体之间是如何通信和协作的。

------

## 我对 ACPs 的理解

ACPs 可以理解为一套**让智能体之间能正常沟通、协作、传输信息**的规则。

就像人类用语言交流一样，智能体也需要统一的协议才能互相理解、配合完成任务。
